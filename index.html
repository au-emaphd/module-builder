Modules

Tab will show all module content organized in accordions, showing prework, classwork, and weekly assignments for each module. Format will be so that content can be copied into D2L HTML pages.

Calendar

Tab will show all learning content organized by week, showing prework, classwork, and weekly assignments for each module with the date. Format will be so that content can be copied into D2L HTML pages.

Database

Tab will show an embed of the Tech Sales Program Base stored in Google Sheets, listing all learning content as individual line items with metadata.

Metrics

Tab will show item lengths, estimate task completion times, and credit hour compliance calculations for the course.
